MulcoTaint is a multi-tag dynamic taint analysis tool via hardware/software co-design.
The prototype of MulcoTaint is implemented on RocketChip and completed the development of the taint calculation engine based on the PHMon coprocessor framework.

The paper is accepted by USENIX Security 2026 'MulcoTaint: Towards Efficient Multi-tag Dynamic Taint Analysis via Hardware/Software Co-design'.

More details are open-sourced in the Zenodo repository https://doi.org/10.5281/zenodo.17939551.
